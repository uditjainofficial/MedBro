# 🩺 MedBro - AI Medical Voice Assistant

An AI-powered voice-based medical assistant that listens to symptoms and provides intelligent responses in real-time.

## 🚀 Features
- 🎤 Voice interaction
- 🧠 AI-powered responses
- 🔐 Authentication (Clerk)
- 🗄️ Database (Neon + Drizzle)

## 🛠️ Tech Stack
- Next.js
- React
- Clerk Auth
- Neon DB
- Drizzle ORM
